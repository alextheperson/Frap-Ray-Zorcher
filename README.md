# Frap-Ray-Zorcher

## Keybinds
|Key       |Use                    |
|----------|-----------------------|
|WASD      |Move                   |
|Arrow Keys|Rotate                 |
|Left Shift|Toggle Going Fast      |
|T         |Toggle Scope           |
|Y         |Fire Laser             |
|U         |Fire Energy Blast      |
|O         |Toggle Hover Stop      |
|P         |Toggle Parachute       |
|G         |Arm/Throw Demise-O-Bomb|
|Tab       |Grab/Repair            |
|Backspace |Toggle Headlamp        |
